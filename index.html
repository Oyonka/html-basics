AN EXAMPLE FOR -br AND -hr TAGS.

In HTML, spaces and new lines are ignored:

My Bonnie lies over the ocean.
My Bonnie lies over the sea.

My Bonnie lies over the ocean.
Oh, bring back my Bonnie to me.

The -br tag is a line break and is used when one needs a new line wiithout a new paragraph.
The -hr tag is a thematic break used to separate content.

HTML STYLES

The style attribute is used to add styles to an element, such as color, font, size, and more.

It has the syntax -tagname style="property:value;"- /tagname -
The property is a CSS property. The value is a CSS value.
We use background-color for background color, color for text colors, font-family for text fonts, font-size for text sizes and text-align for text alignment

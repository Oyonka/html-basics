HTML Quotation and Citation Elements

This are blockquote, q, abbr, address, cite, and bdo

Blockquote
This defines a section that is quoted from another source.Browsers usually indent Blockquote elements.
Example
Here is a quote from Skyprep.com

“Tell me and I forget, teach me and I may remember, involve me and I learn.”
-Benjamin Franklin

Q
This defines a short quotation.Browsers usually insert quotation marks around the q element.
Example
Build a future where people live in harmony with nature.

Abbr
It defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM"
The WHO was founded in 1948.

Address
Defines the contact information for the author/owner of a document or an article. The text in the element usually renders in italic, and browsers will always add a line break before and after the element.
Example

Written by Boke B.
Visit us at:Kiriruru
https://github.com/Oyonka
Box OX-00100
NAIROBI.

Cite
It defines the title of a creative work. Element usually renders in italic.
Example
ANTHROPOLOGY by Boke Marasi. Written in 2015.

Bdo
BDO stands for Bi-Directional Override.
The tag is used to override the current text direction
Example
This text will be written from right to left
